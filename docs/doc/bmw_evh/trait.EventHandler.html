fn set_on_close(&mut self, on_close: OnClose) -> Result<(), Error>

source

fn set_housekeeper(&mut self, housekeeper: HouseKeeper) -> Result<(), Error>

sets the housekeeper callback for this crate::EventHandler. This callback is executed once every crate::EventHandlerConfig::housekeeping_frequency_millis milliseconds. Each thread executes this callback independantly.

source

fn set_on_panic(&mut self, on_panic: OnPanic) -> Result<(), Error>

sets the on panic callback for this crate::EventHandler. This callback is executed whenever a user callback generates a thread panic.
